## Getting Started

Install the module with: `npm install tumblrize`

```js
var tumblrize = require('tumblrize');
tumblrize.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g tumblrize
$ tumblrize --help
$ tumblrize --version
```
